Counting Sort — Implementação em Python
Este projeto apresenta uma implementação passo a passo do Counting Sort, um algoritmo de ordenação eficiente para listas de números inteiros positivos com faixa limitada.

Objetivo:
Este código foi criado com fins didáticos para entender, visualizar e acompanhar internamente o funcionamento do algoritmo Counting Sort, incluindo sua lógica de contagem e distribuição.

Lógica do Algoritmo Counting Sort:
O Counting Sort funciona contando quantas vezes cada valor aparece no array, e depois usando essas contagens para colocar cada elemento na posição correta. 
Ele não compara valores — por isso é mais rápido que algoritmos comuns como Bubble ou Insertion quando os números estão em um intervalo pequeno.
